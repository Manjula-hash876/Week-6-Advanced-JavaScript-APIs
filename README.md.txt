# 🌤 Weather Dashboard Application

A responsive Weather Dashboard web application built using **HTML, CSS, and JavaScript**.  
It fetches real-time weather data using the **OpenWeatherMap API**, supports user preferences, and stores data using **Local Storage** for persistence.

---

## 📌 Project Overview

This project demonstrates advanced JavaScript concepts including:

- Asynchronous JavaScript (async/await)
- REST API integration
- JSON data handling
- Local Storage & Session Storage
- Modular JavaScript architecture
- Error handling & debugging
- Responsive UI design

---

## 🚀 Features

### 🌍 Core Features
- Search weather by city name
- Display current weather conditions
- Show 5-day weather forecast
- Real-time API data fetching

### ⭐ User Features
- Save favorite cities
- Store user preferences (units, theme, default city)
- Persistent data using Local Storage
- Session-based temporary data handling

### ⚡ Advanced Features
- Weather data caching (performance optimization)
- Debounced search input
- Loading states and error handling
- Responsive design for all devices

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (ES6+)
- OpenWeatherMap API
- Local Storage API
- Fetch API (async/await)
